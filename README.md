
# dotfile-i3-wm-cachyos-Archlinux

![screenshot](screenshot.png)

## Instalación
```bash
git clone https://github.com/Panamiguel77/dotfile-i3-wm-cachyos-Archlinux.git
cd dotfile-i3-wm-cachyos-Archlinux
chmod +x setup.sh && ./setup.sh
```

## Programas
- WM: i3
- Bar: Polybar
- Launcher: Rofi
- Terminal: Kitty
- Compositor: Picom
